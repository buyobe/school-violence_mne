Quick System Guide

Welcome to the School Violence Monitoring System. This dashboard serves as a quick manual:

  • Admins: Manage users, backup/restore data, configure system settings.
  • DataEntry: Upload Excel data and manage entries.
  • Viewers: Explore indicators, visualization, and reports.

For a full manual, see the System Documentation (PDF).

User Management

Admins can add, edit, delete users and assign roles.

  1. Go to View Users to see all accounts.
  2. Click Add User to register a new account.
  3. Use Edit to change details or assign roles.
  4. Use Delete to remove accounts (with safeguards).

Data Backup

Keep your system safe by backing up data regularly.

  1. Click Backup Database to download a JSON file.
  2. Store backups securely outside the system.
  3. Use Restore Database to reload saved data.
